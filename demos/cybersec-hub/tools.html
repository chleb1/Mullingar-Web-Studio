Network Scanning & Enumeration

Tools for discovering hosts, open ports, running services, and network topology.

🗺 Nmap
The definitive network scanner. Host discovery, port scanning, service/version detection, OS fingerprinting, and NSE scripting engine. Used in almost every engagement.
Free / Open SourceLinux · Windows · MacBeginner
⚡ Masscan
The world's fastest TCP port scanner. Sends up to 10 million packets/second. Excellent for scanning large address ranges quickly. Less feature-rich than Nmap.
Free / Open SourceLinuxIntermediate
🔭 Shodan
Search engine for internet-connected devices. Find exposed servers, cameras, ICS systems, and services globally. Indispensable for OSINT and external attack surface mapping.
Free / Premium APIWeb-basedBeginner
🕷 Enum4linux
Enumerates Windows/Samba systems — users, shares, password policies, group memberships. Essential for Active Directory reconnaissance.
FreeLinuxIntermediate
🗂 CrackMapExec (CME)
Swiss army knife for Windows/Active Directory assessments. SMB enumeration, credential testing, command execution, and lateral movement across networks.
FreeLinuxIntermediate
🔍 Netdiscover
Active/passive ARP scanner for local network host discovery. Great for quickly mapping a LAN during internal assessments.
FreeLinuxBeginner
🦈 Wireshark
The world's most popular network protocol analyser. Capture and deep inspect network traffic with 3000+ protocol dissectors. Essential for network forensics and analysis.
FreeCross-platformBeginner
🌐 theHarvester
OSINT tool for gathering email addresses, domains, hosts, employee names, and open ports from public sources. Great for pre-engagement recon.
FreePythonBeginner
🔎 Recon-ng
Full-featured web reconnaissance framework written in Python. Module-based architecture similar to Metasploit. Automates OSINT collection from dozens of sources.
FreePythonIntermediate
📡 Responder
LLMNR, NBT-NS, and MDNS poisoner. Captures NTLMv1/v2 hashes from Windows hosts on the local network. Extremely effective in internal assessments.

Exploitation Frameworks & C2

💥 Metasploit Framework
The world's most widely used penetration testing framework. 2,000+ exploit modules, payloads, auxiliary modules, post-exploitation. Essential for any security professional.
Free / ProLinux · WindowsIntermediate
📡 Cobalt Strike
Commercial adversary simulation platform. Beacon C2, malleable profiles, aggressor scripts. The industry standard for red teams. Expensive but widely used in APT simulation.
Commercial ($3,500/yr)Linux · WindowsPro
🌐 Sliver
Open-source C2 framework from BishopFox. TLS, HTTP/S, DNS, and WireGuard C2 channels. Modern Cobalt Strike alternative with active development.
FreeGoPro
👹 Havoc Framework
Modern red team C2 framework with Demon agent. Supports BOFs, sleep obfuscation, process injection, and custom profiles. Growing alternative to Cobalt Strike.
FreeC / PythonPro
🩸 BloodHound
Graph-based Active Directory attack path visualisation. Find shortest path to Domain Admin, identify attack paths, and prioritise remediations. With SharpHound collector.
FreeWindowsIntermediate
🥷 Mimikatz
Extracts credentials from Windows memory — NTLM hashes, plaintext passwords, Kerberos tickets. The most impactful Windows credential tool ever created.
FreeWindowsPro
⚙️ Impacket
Python library for working with network protocols. psexec.py, wmiexec.py, secretsdump.py, GetUserSPNs.py — essential for AD post-exploitation.
FreePythonPro
😈 Evil-WinRM
WinRM shell for pentesting and post-exploitation. Supports file upload/download, DLL load, log avoidance, and pass-the-hash authentication.
FreeRubyIntermediate

Password Auditing & Cracking

🔑 Hashcat
GPU-accelerated password recovery tool. Supports 350+ hash types. Multiple attack modes: dictionary, brute force, mask, rule-based, combination. The fastest cracker available.
FreeCross-platformIntermediate
🗡 John the Ripper
Classic password cracker. Excellent for cracking Unix/Linux shadow files, ZIP archives, PDF passwords, and many other formats. CPU-based, very flexible.
FreeCross-platformBeginner
💧 Hydra
Fast online password brute-forcing tool. Supports 50+ protocols: SSH, RDP, FTP, HTTP, SMTP, SMB, and more. Parallelised for speed.
FreeLinuxIntermediate
🔓 Medusa
Fast, parallel, modular login brute-forcer. Alternative to Hydra. Supports many protocols with a clean module-based architecture.
FreeLinuxIntermediate
📚 SecLists
The most comprehensive collection of wordlists for security testing — passwords, usernames, URLs, fuzzing payloads, and more. Essential companion for any tool.
FreeCross-platformBeginner
🌊 Spray
Password spraying tool targeting Active Directory environments. Sprays a single password against many accounts to avoid lockouts.
FreeGoIntermediate

Malware Analysis & Reverse Engineering

👻 Ghidra
NSA's free, open-source reverse engineering suite. Disassembler, decompiler, scripting (Python/Java), multi-architecture support. Rivals IDA Pro at no cost.
Free / NSACross-platformIntermediate
🔭 IDA Pro
Industry-standard interactive disassembler and debugger. 30+ years of development. Hex-Rays decompiler is the gold standard. Very expensive but irreplaceable in serious RE work.
Commercial (~$1,700+)Pro
🐛 x64dbg
Open-source x64/x32 debugger for Windows. Modern interface, plugin ecosystem, excellent for dynamic malware analysis and patch analysis.
FreeWindowsIntermediate
🥚 Cuckoo Sandbox
Automated malware analysis sandbox. Execute suspicious files and observe behaviour — network connections, file system changes, registry modifications, API calls.
FreeLinuxIntermediate
🌐 ANY.RUN
Interactive online malware sandbox. Run suspicious files in a browser, interact with them live, and see behavioural analysis instantly. Free tier available.
Free / PremiumWeb-basedBeginner
🔥 FLARE VM
Windows-based security distribution for malware analysis and reverse engineering. Installs 100+ RE tools automatically on Windows.
Free / FireEyeWindowsIntermediate
🦠 VirusTotal
Multi-AV file and URL scanner. 70+ antivirus engines, YARA rules, behavioural analysis, and community comments. First stop for analysing suspicious files.
Free / EnterpriseWeb-basedBeginner
📝 YARA
Pattern matching tool for malware identification. Write rules to detect malware families by their byte patterns, strings, or structural characteristics. Used by AV and threat intel teams.
FreeCross-platformPro
