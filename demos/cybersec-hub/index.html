Learning Paths

Choose Your Direction

Structured paths to get from where you are to where you want to be.

🛡️

Defender Path

SOC Analyst → Threat Hunter → Security Engineer → CISO

  • CompTIA Security+ (6 months)
  • Build home SIEM lab
  • CySA+ or GSOC certification
  • Specialize in cloud or forensics
  • Pursue CISSP at senior level
💀

Attacker Path

Jr. Penetration Tester → Senior Pentester → Red Team Lead

  • CompTIA Security+ or eJPT
  • TryHackMe / HackTheBox
  • Learn Python + Bash scripting
  • OSCP certification
  • Specialize (web, AD, mobile)
🔍

Research Path

Security Researcher → Malware Analyst → Vulnerability Researcher

  • Strong C/C++ and Assembly
  • Reverse engineering fundamentals
  • Bug bounty hunting
  • GREM / GXPN certifications
  • CVE discovery & disclosure
